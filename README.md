# Math Expression Evaluator v0.4
Evaluates math expressions. Pretty self explanatory.

## How to Use and Run
(assuming you are in the project/repo folder)
```
cd src
py main.py "<math_expression>"
```

For example, if you wanted to evaluate `1 + 2`, then you would do:
```
cd src
py main.py "1 + 2"
```
